# Ren'Ts

**在 Windows / macOS / Linux / iOS / Android 上游玩 Ren'Py 视觉小说的阅读器**


> ⚠️ 目前为早期测试版，欢迎试用并反馈问题。

---

## 这是什么

很多 Ren'Py 视觉小说只有 PC 版，没有手机版，安装也比较繁琐。Ren'Ts 是一个桌面阅读器，让你可以把 Ren'Py 游戏的脚本文件导入进来，直接在电脑上阅读游玩，不需要安装 Python 或 Ren'Py 本体。

---

## 下载

前往 [Releases 页面](../../releases) 下载最新版本：

- **macOS**：下载 `.dmg` 文件，打开后将 Ren'Ts 拖入应用程序文件夹
- **Windows**：下载 `.exe` 安装包，双击安装
- **Linux**：下载 `.AppImage` 文件，添加可执行权限后直接运行
- **iOS**：可以直接使用 [Web 版](#)，用 Safari 打开网址即可，无需安装。但受限于 iOS 的浏览器限制，Web 版**不支持自动存档、无法使用脚本转换工具**，且需要提前在电脑上转换好脚本再导入。想体验完整功能需自行编译安装（见下方说明）
- **Android**：下载 `.apk` 内测包，安装时允许"来自未知来源的应用"即可

### macOS 打开时提示"无法验证开发者"？

这是正常现象，因为目前版本尚未购买苹果开发者证书。解决方法：

1. 右键点击 Ren'Ts 图标
2. 选择「打开」
3. 在弹出的对话框中再次点击「打开」

之后就可以正常使用了，只需操作一次。

## 使用方法

### 你需要准备什么

游玩一款游戏需要两样东西：

1. **游戏的脚本文件**（`.rpy` 文件）— 见下方「如何获取脚本文件」
2. **游戏的素材文件**（图片、音频等）— 同样在游戏安装目录中

> Ren'Ts 本身不包含任何游戏内容，需要你自己准备。请确认你拥有对应游戏的合法使用权。

### 如何获取脚本文件

大多数 Ren'Py 游戏的文件都可以直接在安装目录中找到，不需要任何破解或特殊工具。

**Windows / macOS / Linux：**

1. 找到游戏的安装目录（Steam 游戏可右键 → 管理 → 浏览本地文件）
2. 打开其中的 `game/` 文件夹
3. 里面的 `.rpy` 文件就是脚本，图片和音频也在同一目录下

**如果只有 `.rpyc` 文件而没有 `.rpy` 文件：**

部分游戏只发布了编译后的 `.rpyc` 文件，这种情况目前暂不支持，欢迎提 Issue 告知具体游戏名称。

### 第一步：找到游戏文件

在你电脑上找到 Ren'Py 游戏的安装目录，里面通常有一个 `game/` 文件夹，其中包含 `.rpy` 脚本文件以及 `images/`、`audio/` 等子文件夹。

### 第二步：转换脚本

1. 打开 Ren'Ts
2. 点击左上角的**工具图标**
3. 在「游戏目录」一栏填入游戏的 `game/` 文件夹路径
4. 按照提示填写其他信息
5. 点击「转换」，等待完成即可

### 第三步：开始游玩

转换完成后，主界面会要求你选择素材文件夹：

- 如果转换时没有设置输出路径（默认原地转换），直接选游戏原本的 `game/` 文件夹即可
- 如果设置了输出路径，选择对应的输出文件夹

## 注意事项

- **部分语法不支持**：Ren'Ts 目前不支持全部 Ren'Py 语法，复杂脚本可能无法正常运行，欢迎反馈具体游戏名称
- **素材需自备**：图片、音频等素材文件不会被打包进转换结果，需要保留原游戏目录
- **iOS 需自行编译**：受限于苹果开发者账号费用，暂无法直接分发 iOS 安装包

- **特殊跳转补丁（权宜之计）**：某些游戏在进入日记 / minigame / 特殊 UI 流程时，会使用复杂的原生界面与脚本交互（例如 `journal_start`、`foreplay` 等标签）。当前引擎无法在内置阅读器中完整复现这些 UI/minigame。为避免游戏在转换后直接卡住或误进入无法处理的流程，转换器中加入了一个临时的不优雅补丁：当遇到诸如 `jump journal_start;` 或 `jump foreplay;` 这样的跳转时，会把它们改写为 `jump label_afterjournal;` / `jump label_afterforeplay;`（与脚本里预先设置的 `label_after...` 变量配合使用），从而直接跳到后续流程，绕过需要原生 UI 的入口。

  该补丁可以让多数剧情顺利继续，但它是一个工程折衷方案，存在局限性和潜在副作用（例如可能跳过原本应执行的副作用或未覆盖所有同类命名）。如果你对某款游戏的这类片段有更好的处理建议，或遇到因该补丁导致的问题，请在 Issue 中附上具体游戏和位置（方便复现），我们会评估更稳健的解决方案（例如在转换层引入专门的 `enter_minigame` step，或由 UI 层提供可配置的挂钩）。

## 常见问题

**Q：支持哪些游戏？**  
A：理论上支持大多数使用标准 Ren'Py 语法的游戏，但目前仍有部分高级语法尚未实现。如果你的游戏无法运行，欢迎提 Issue。

**Q：iOS 怎么安装？**  
A：推荐先试试 [Web 版](#)，用 Safari 打开网址即可，不需要安装任何东西。不过由于 iOS 强制使用 WebKit 引擎，Web 版有以下限制：不支持自动存档、无法使用内置的脚本转换工具、部分文件读取功能受限。需要先在电脑上转换好脚本，再手动导入。如果想要完整体验，需要自己用 Xcode 编译安装，对普通用户门槛较高。由于上架 App Store 需要每年 ¥688 的苹果开发者账号，暂时无法直接分发安装包，如果你愿意赞助欢迎联系 👀

**Q：转换出来的文件可以分享给别人吗？**  
A：转换文件本身只是格式转换，版权仍属于原游戏作者，请勿在未获授权的情况下分发。
